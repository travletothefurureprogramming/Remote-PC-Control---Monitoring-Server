# Remote PC Control & Monitoring Server
### A web-based dashboard designed to monitor system performance and manage PC operations remotely. 

<img width="1918" height="906" alt="image" src="https://github.com/user-attachments/assets/1ac9a423-f4ab-41ae-a36b-a78485c9ddb2" />

## Features

The Remote PC Control & Monitoring Server includes the following key features:

* System Monitoring
* Network Statistics
* System Uptime
* Power Management
* Media Control
* Process Tracking
* Battery Status
* Responsive UI

## Tech Stack

* Languages: **Python 3.11**, **HTML5**
* Environment: The package suppourt **Windows**. But you can run in any Platform with the Source Code.
* Hardware: I have tested it in a HP Victus **Gaming Laptop**
